# About This Project

This application was developed as part of the **ft_transcendence** project at 42.

The goal of the project is to design and implement a complete web application featuring real-time interaction, user management, and modern web technologies.

## Key features

- User accounts and authentication
- Multiplayer gameplay
- Real-time communication
- Persistent user data

## Technology stack

- **Frontend:** React / Next.js
- **Backend:** Node.js with a web framework
- **Database:** PostgreSQL
- **Real-time features:** WebSockets

This project is intended as a learning experience and not as a commercial service.
